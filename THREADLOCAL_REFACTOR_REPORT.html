Module-home decision

Extend engine/context.py (not a new engine/reqctx.py). Rationale: engine/context.py is

already the cycle-free home of _thread_local (imports only stdlib, never import brain), and is

already the import target for all three spellings (engine._thread_local, brain._thread_local,

from engine.context import _thread_local). Putting RequestContext + the ContextVar + the shim

here keeps the single-instance-identity invariant trivially true and adds no new import edges.
