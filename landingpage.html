What it does

Everything an agent needs,
none of the ceremony.

Built around a single Python server with a native agentic loop. No SDK sidecars, no hidden magic — just tools, agents, and a cockpit that stays out of your way.

Projects

Claude.ai-style workspaces with custom instructions, ingested documents, markdown notes with AI-assisted editing, and per-project chat scoping. Archive, search, and filter from a card grid.

Chat history

Every session is persisted with LLM-generated one-line summaries, full-text search across titles and content, archive / delete controls, and restore-on-click — the sidebar remembers everything.

Attachments

Drop images, PDFs, or any file into the composer. Vision-capable models see them natively; text models get disk-handoff plus read_document across PDF, DOCX, XLSX, PPTX, CSV, and images.

References panel

Every web citation from exa_search and web_fetch becomes a source card with favicon, screenshot preview via Microlink, and click-through — Le Chat-style grounding, built in.

Artifacts

Files an agent writes under artifacts/ are versioned in SQLite and opened in a type-aware right panel — code with syntax highlighting, HTML in iframe, SVG inline, images, rendered markdown — with copy and download.

Prompt refinement

Fire an LLM pass over whatever you've typed to sharpen wording, add structure, and fold in session context — all before the message is even sent. One keystroke, zero roleplay.

Next-prompt suggestions

Claude Code-style ghost text in the composer. After each reply, a tiny LLM call predicts your next message and offers it as dimmed placeholder — Tab accepts, Esc dismisses. Reuses the session's prompt cache, near-free.

Multi-agent system

Every agent has its own soul.md, model, tools, skills, and MCP servers. Organize them into teams with hierarchical delegation, quick-switch from the sidebar, assign per-model routing.

MemPalace memory

Long-term memory is a first-class MCP server. Store, recall, and consolidate through a clean tool surface — no engine-level magic, no indexer daemons to babysit.

Scheduler

Cron-style recurring tasks, one-shot jobs, and autonomous loops. Each task gets its own agent, model, context, and result history — with a live dashboard of what's running right now.

Skills & MCP

Load 7000+ OpenClaw-compatible skills on demand. Wire any MCP server per-agent via mcp.json — stdio, SSE, or HTTP — and every MCP tool is dedup-protected and budget-aware.

Lossless context

DAG-based hierarchical summarization keeps you inside the token budget without losing what the model already saw. Drill back into any compacted message with context_search, context_detail, and context_recall.

Recent milestones

Shipping, often.

Eight major versions so far. The most recent rewrite moved memory out of the engine and into an MCP server — and stripped 30k lines of dead code along the way.

v8.0.0 · 2026-04-15

MemPalace migration

Memory moved fully to an external MCP server. Removed built-in memory tools, QMD daemon, knowledge graph, autodream consolidation, and all /v1/agents/{id}/memories endpoints. ~2,400 lines of code deleted.

v7.1.0 · 2026-04-08

Next-prompt suggestions

Claude Code-style ghost text in the composer — predicted next user message, cache-reused, dismiss-on-type.

v7.0.0 · 2026-04-02

Native agentic loop restored

PI and Anthropic SDK sidecars removed. Streaming is now raw socket SSE end to end, no anyio buffering.

v5.3.0 · 2026-03-31

Claude.ai-style web UI rewrite

Sidebar + multi-view layout, Anthropic fonts, warm themes. Tool call blocks with full args persist across reloads.

v4.0.0 · 2026-03-23

Universal file intelligence + code graph

Read/write/edit across XLSX, PPTX, CSV, images, SVG. Tree-sitter AST code graph in 14 languages with blast-radius analysis.

v1.0.0 · 2026-03-14

Client-server architecture

Telegram bot, background tasks, scheduler. The moment this stopped being a script and became a platform.
