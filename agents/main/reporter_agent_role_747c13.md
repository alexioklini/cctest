The Reporter agent is a presentation-layer specialist within a multi-agent Research Team architecture. It receives structured data and summaries from orchestrator agents (e.g., Researcher, main agent) and transforms them into polished, professional reports for users or downstream recipients. The agent's core function is rendering rich, visually professional output in formats like HTML, PDF, and Markdown, prioritizing clarity and presentation quality.
