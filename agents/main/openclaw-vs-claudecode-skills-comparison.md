Key findings: OpenClaw is a general-purpose autonomous agent (open-source, self-hosted, 40+ messaging integrations, ClawHub marketplace with 5700+ skills but security concerns). Claude Code is a purpose-built coding agent (Anthropic-managed, terminal/IDE, rich skill structure with subagent isolation, tool permissions). Both use SKILL.md files but differ fundamentally in execution model, security, extensibility, and target use case. They are complementary, not competing tools.
